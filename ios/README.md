# PSPDFKit Flutter for iOS

## Requirements

 - The latest [Xcode](https://developer.apple.com/xcode/)
 - PSPDFKit 10.2.0 for iOS or later
 - Flutter 1.22.6 or later
 - CocoaPods 1.10.1 or later (Update cocoapods with: `gem install cocoapods`)
